💻 Requisitos Mínimos del Sistema

🖥️ Hardware Mínimo

  • Procesador: Intel Core i5 o AMD Ryzen 5 (8va gen o superior)
  • RAM: 8 GB mínimo (16 GB recomendado para CAD/BIM)
  • Disco Duro: 500 GB disponibles (SSD recomendado)
  • Tarjeta Gráfica: 2 GB VRAM (4 GB para diseño 3D)
  • Pantalla: Resolución mínima 1366x768 (1920x1080 recomendado)

💿 Software Base

  • Sistema Operativo: Windows 10/11 64-bit o macOS 12+
  • Navegador: Chrome, Firefox o Edge actualizado
  • Internet: Conexión estable mínimo 10 Mbps
  • Antivirus: Actualizado (excepto durante instalación)
  • Espacio: Varía según software (50-150 GB)

❓ Preguntas Frecuentes

¿Puedo instalar el software en mi computadora personal?

+

Sí, las licencias educativas permiten instalación en equipos personales, pero solo para uso académico. Debes activarlas con tu correo institucional @senati.pe.

¿Qué hago si no puedo activar la licencia?

+

Verifica que estás usando tu correo @senati.pe. Si el problema persiste, contacta a soportevirtual@senati.pe con tu código de estudiante y captura de pantalla del error.

¿Las licencias tienen fecha de expiración?

+

Sí, las licencias son válidas mientras seas estudiante activo en SENATI. Se renuevan automáticamente cada semestre académico.

¿Puedo instalar el software en más de un equipo?

+

Depende del software. La mayoría permite 2 instalaciones activas. Consulta los términos específicos en el manual de cada programa.

¿Hay versiones para Mac/Linux?

+

Algunos programas tienen versiones multiplataforma. Verifica en la descripción de cada software o contacta a soporte técnico para confirmar compatibilidad.

¿Dónde encuentro las claves de activación?

+

Las claves se envían automáticamente a tu correo institucional después de descargar el software. También puedes solicitarlas en la oficina de sistemas de tu sede.
